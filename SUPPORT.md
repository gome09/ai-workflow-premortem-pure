## Project Status

This is an **open-source local-preview project**. It is intended for personal use and trusted small-team internal preview.

It is **NOT** production-ready, SaaS-ready, or multi-tenant-ready.

## Support Policy

Support is **best-effort**. This project is maintained by volunteers and is not backed by a commercial support agreement.

- Issues will be reviewed as time permits
- Pull requests are welcome but may take time to review
- There is no guaranteed response time

---

## Security Issues

**Do not open public issues for security vulnerabilities.**

Please follow the instructions in [SECURITY.md](SECURITY.md) to report security issues privately.

---

## What We Can Help With

- Setup and configuration issues on localhost
- Bug reports with clear reproduction steps
- Documentation improvements
- Feature requests aligned with the project scope

## What We Cannot Help With

- Production deployment assistance
- Multi-tenant or enterprise configuration
- Custom development or consulting
- Issues arising from public internet exposure of the service
