📌 关于本 Demo

本 Demo 是 AI 工作流风险预评估系统的离线可交互版本，所有数据来自真实项目实跑快照。

  • 离线模式配置：LLM_MODE=mock · STORAGE_BACKEND=sqlite · DEFAULT_SCENARIO_ID=generic_rag_demo · WORKFLOW_EXECUTION_MODE=single_step
  • 无需后端服务、无需 API Key、无需联网，使用 Mock 数据模拟真实系统的四阶段工作流
  • 状态机、门禁规则、部署决策引擎均与真实项目一致

真实工程版本基于 FastAPI + SQLite，支持 Mock LLM、四阶段工作流、风险自适应门禁、证据核验、Eval 评估体系、Red Team、报告导出等完整能力。
